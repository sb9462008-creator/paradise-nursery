# Paradise Nursery Shopping Application

## Project Description

Paradise Nursery is an online plant shop that allows users to browse and purchase a variety of houseplants. This is a dynamic React-based web application built with Redux for state management.

## Features

- Browse plants organized by category (Air Purifying, Aromatic, Foliage)
- Add plants to shopping cart
- View cart with item quantities and total cost
- Increase or decrease item quantities
- Remove items from cart
- Responsive navigation bar with cart item count

## Technologies Used

- React.js
- Redux Toolkit
- CSS3
- JavaScript (ES6+)
